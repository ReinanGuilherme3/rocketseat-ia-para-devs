# 📖 Sobre

Este repositório reúne todo o material produzido durante o curso **IA para Devs**, incluindo exemplos práticos, projetos, laboratórios, experimentos e anotações.

O curso apresenta uma visão completa do desenvolvimento de aplicações utilizando Inteligência Artificial Generativa, cobrindo desde conceitos fundamentais até arquiteturas modernas baseadas em LLMs, RAG e Agentes de IA.

Todo o conteúdo desenvolvido aqui possui finalidade de estudo e servirá como base para futuras implementações utilizando IA em aplicações reais.
